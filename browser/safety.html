# 前端安全

# XSS

跨站脚本攻击，攻击者通过注入非法的HTML标签或js代码，使之在用户浏览器上运行，利用这些恶意脚本，控制用户浏览器，获取用户信息等。

# XSS攻击分类

存储型

反射性

DOM型

解决方法

  1. 输入格式校验（数字，电话，URL等）
  2. 输入内容进行转义
  3. httpOnly

# CSRF

跨站请求伪造，网站中的一些提交行为被黑客利用，在用户访问黑客的网站时进行的操作会被操作到其他网站上。

解决办法

  1. 重要操作使用验证码
  2. 尽量使用POST
  3. 可以使用token来验证

ps:该攻击下，httpOnly无法防住，因为CSRF是伪装成用户发送请求，浏览器会自动携带cookie。
