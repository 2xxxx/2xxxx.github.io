# OSI七层模型  
OSI(Open System Interconnection,开放系统互联)七层模型把网络分为了7层，依次是：  
|层次        |主要协议                 |备注       |
|------------|------------------------|-----------|
|应用层       |HTTP、FTP、TELNET、SMTP |  |
|表示层      |ASCII、SSL/TLS          |相当于翻译|
|会话层      |ADSP、RPC               |维护通信 |
|传输层      |TCP、UDP                |负责两节点间的数据传输 |
|网络层      |IP、ICMP                |地址管理、路由选择|
|数据链路层  |HDLC、PPP、SLIP         |           | 
|物理层     |RS232C、RS432            |            |
